# Gestão de Cinema

Este projeto consiste no desenvolvimento de uma API REST para um Mini-ERP voltado à gestão de Cinema. Construída com Spring Boot e arquitetura MVC, a aplicação adota o formato XML para troca de dados e foca no controle operacional completo: desde o cadastro de filmes e salas até a validação técnica de exibição (homologação), agendamento de sessões com geração automática de estoque de assentos e venda de ingressos.
